# Example

To run the script use:
```
python3 classy.py --bnet n7s3.bnet --classifier "v4 & !v6"
```


Syntax for `.bnet` files and classifier definition is
 * `|` means disjunction 
 * `&` means conjunction
 * `!` means negation


# Requirements

The main requirement is SAGE:
 * https://www.sagemath.org/
